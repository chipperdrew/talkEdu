State of the Website (12/20/2013)

Thank you for taking the time to read this bland yet useful page. The purpose of this page is to inform visitors of upcoming changes that will be made to the site.


Rationale

It comes as no surprise that this site is basically dead. If you have come here to post, please do!! Even if only one person reads your thoughts, you still have made a difference.

Unfortunately, as a full-time college student, I simply do not have the time, money, marketing ability, and general effort to try to turn this site into a lively forum for discussion (as much as I wish I could. Thus, I have decided to make a few changes to the site, outlined below.


Changes and Resulting Consequences

  • Fewer resources will be devoted to the processing power behind the website
    • Consequences: If you are the first visitor to the website in the past hour, the first YouTalkEdu page accessed will take many seconds to load. All pages accessed after the initial page should load with normal speed.
  • To save money, I plan on limiting the size of the database used for the site.
    • Consequences: Pages may take a little extra time to load since caching will be disabled. This should be minimal in effect.
  • There will be no security changes made. This site will remain under SSL protection.


Once again, I apologize to all users and future visitors for any inconviences these changes have caused. To help remedy any pains, here is a link to a funny video. Enjoy.
