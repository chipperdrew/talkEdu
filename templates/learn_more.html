What is YouTalkEdu?

YouTalkEdu is a free-to-use website geared towards bettering the current educational system. Anyone can browse around the content here, however only those with an account can post, comment, or vote on the content. Registering is extremely painful and not recommended, but if you wish to do so you may register here.


General guidelines:

  • Don't post spam. Ever.
  • Don't slander specific people or schools.
  • No foul language.

Click on the tabs to the left to learn more about YouTalkEdu!

What's the purpose?

Great question! The different types were created to get a "feel" of the thoughts and opinions from each camp.

For example, there may be a large amount of content from teachers about the costs of buying student supplies, presenting an issue to the other user types (and the rest of the community) that may have previously gone unconsidered.


What's with the colors?

Another great question! Color can be used to stimulate learning and is beneficial in bringing out creativity (you can read more about this in the rather uncolorful eHow article located here).

Also, the voting system uses the colors for its display. If you haven't already, check out the voting tab on the left to learn more (I promise it's super interesting).

PIQS - A very unclever abbreviation for the content pages.

PIQS is an abbreviation for the content pages on the site. It stands for:


{{ page_type_dict.0.1 }} - Discuss any problems you see in the current educational system here

No specifics!! Phrases such as "Mr. Doe should be fired" or "Blah High School is failing students" WILL get you banned.


{{ page_type_dict.1.1 }} - Discuss any ideas which may benefit the educational system here

If you've tried anything to improve education in your school/community, I STRONGLY encourage you mention it here.


{{ page_type_dict.2.1 }} - Discuss any educational questions worth considering here

Once again, specific questions such as "Why is Mr. Doe so bad?" WILL get you banned.


{{ page_type_dict.3.1 }} - Discuss any thoughts you have about YouTalkEdu

This site was built for the purposes of improving education. If you have any ideas about how to better the discussion, or just simply want to voice your thoughts on what you like/dislike/want to see, please post here.
If you have found any security issues with the site, PLEASE do not post them here. Email those to me and I will be forever grateful.

A voting system with a purpose.

Each post/comment has a vote chart like the following:



  • Each vote tube corresponds to a given user type (Red - Students, Blue - Parents, etc).

  • The tubes are percentage based -- they indicate the percentage of votes that are "up" votes by each user type.

  • The black line across the vote chart represents the average "up" percentage for all users.

  • Underneath each chart is the decimal value corresponding to the value of the overall "up" percentage,
    and the total number of votes on the post.
